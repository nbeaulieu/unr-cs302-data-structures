# Accessibility Statement

## Commitment to Accessibility

This website is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Standards

This website aims to conform to:
- **WCAG 2.1 Level AA** (Web Content Accessibility Guidelines)
- **Section 508** standards (for U.S. federal agencies and institutions receiving federal funding)

## Accessibility Features

### Keyboard Navigation
- All interactive elements are keyboard accessible
- Skip to main content link available (press Tab on page load)
- Logical tab order throughout the site
- Keyboard shortcuts documented for interactive visualizations

### Screen Reader Support
- Semantic HTML structure with proper headings (h1, h2)
- ARIA labels and roles for navigation and interactive elements
- Descriptive link text and aria-labels
- Live regions for dynamic content updates

### Visual Accessibility
- High contrast color scheme
- Focus indicators on all interactive elements
- Text resizable up to 200% without loss of functionality
- Responsive design works on all screen sizes

### Motion and Animation
- Respects `prefers-reduced-motion` user preference
- Animations can be paused in visualizations
- No auto-playing content

### Interactive Visualizations
- Keyboard navigation supported (Arrow keys, Space, R for reset)
- ARIA labels on all controls
- Step-by-step mode available
- Speed controls for animations

## Known Issues

We are aware of the following accessibility issues and are working to address them:

1. **Color Contrast**: Some text colors may not meet WCAG AA standards in all contexts. We are reviewing and updating color combinations.

2. **Visualization Descriptions**: Some complex visualizations may benefit from additional text descriptions. We are working to add more detailed descriptions.
